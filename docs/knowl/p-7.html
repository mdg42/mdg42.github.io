To write small sets, we use curly braces. For example, we could write a set of Natural numbers as \(\{1, 2, 5, 7\}\text{.}\) While this works well for small sets, clearly we cannot write all the Natural numbers this way. Additionally, writing "the Natural numbers" would prove difficult in many contexts where they are used, so we use the symbol \(\mathbb{N}\) to represent the set of Natural numbers.
